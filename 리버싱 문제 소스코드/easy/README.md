# Easy 리버싱 문제들

## 문제 목록

1. **problem_01** - 간단한 문자열 비교
2. **problem_02** - 숫자 비교 (42)
3. **problem_03** - 두 수의 합 (100)
4. **problem_04** - 문자열 길이와 특정 문자 검사
5. **problem_05** - 곱셈 연산 (15 * 25)
6. **problem_06** - 대문자 검사
7. **problem_07** - 배열 검색 (홀수)
8. **problem_08** - 짝수 양수 검사
9. **problem_09** - 관리자 접근 검사
10. **problem_10** - 숫자 맞추기 (7)

## 힌트
- 정적 분석으로 충분히 해결 가능
- 문자열과 숫자 비교가 주요 패턴
- 디스어셈블러나 헥스 에디터 활용
